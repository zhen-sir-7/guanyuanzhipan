原产地规则指南

什么是原产地规则？

原产地规则是各国为了确定货物“原产国”而制定的法律法规。在国际贸易中，原产地直接决定关税税率、配额适用以及是否享受贸易协定优惠。本系统支持对RCEP、CAI等协定的原产地规则进行自动判定。

完全获得 (WO)

指货物完全在某一成员方境内生长、开采、收获或利用该国原料生产。常见于农产品、矿产品、活体动物等。

示例： 在中国种植并收获的小麦，直接出口至日本，可视为完全获得产品。
税则归类改变 (CTC)

指非原产材料经过加工后，其HS编码的前四位（品目）或前六位（子目）发生变化。通常分为：

  • 章改变 (CC, 2位): HS编码前2位变化
  • 品目改变 (CTH, 4位): HS编码前4位变化
  • 子目改变 (CTSH, 6位): HS编码前6位变化
示例： 进口税则号8708的零部件，在中国组装成税则号8703的整车，发生品目改变，满足CTC规则。
区域价值成分 (RVC)

指产品中来自成员方的价值比例。通常有两种计算方法：

  • FOB价法： (FOB价 - 非原产材料价格) / FOB价 × 100%
  • 净成本法： (净成本 - 非原产材料价格) / 净成本 × 100%

不同贸易协定对RVC门槛要求不同，例如RCEP要求≥40%（FOB法）或≥35%（净成本法）。

示例： 产品FOB价100美元，非原产材料30美元，则RVC = (100-30)/100 = 70%，满足RCEP要求。
特定加工工序

部分产品要求必须经过特定的生产工艺才能获得原产资格，例如化学反应、纺织品的特定整理工序等。

RCEP原产地规则特点
  • 允许使用累积规则：成员方境内的生产均可累积
  • 可选RVC≥40%（FOB法）或≥35%（净成本法）
  • 部分产品可采用CTC或特定加工工序
  • 微小含量规则：非原产材料价值不超过10%FOB价不影响原产地
如何使用本系统判定原产地？

1. 在“产品信息管理”中添加产品及原料信息。
2. 进入“原产地判定”，选择产品并指定贸易规则（如RCEP）。
3. 系统将根据HS编码匹配对应的判定步骤（WO、CTC、RVC等），并自动计算。
4. 若通过所有步骤，系统将提供原产地证明模板。
